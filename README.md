# COSTING-SYSTEM-PYTHON

# سیستم هزینه‌یابی محصولات

این پروژه یک نرم‌افزار دسکتاپی ساده با **Python + Tkinter + SQLite** است که برای مدیریت محصولات و مواد اولیه، محاسبه قیمت تمام‌شده و خروجی گرفتن به اکسل طراحی شده است.  

---

## امکانات
- مدیریت محصولات (ثبت، مشاهده، حذف)
- مدیریت مواد اولیه (ثبت، ویرایش، حذف)
- جست‌وجوی محصولات
- محاسبه قیمت تمام‌شده با احتساب:
  - مواد اولیه
  - دستمزد
  - هزینه برق، گاز، آب
  - هزینه‌های سربار
- گزارش هزینه و نمایش در جدول
- خروجی به **Excel** برای تحلیل بیشتر

---

## تکنولوژی‌ها
- Python 3
- Tkinter (رابط گرافیکی)
- SQLite (ذخیره‌سازی داده)
- Pandas (خروجی اکسل)
- tkcalendar (انتخاب تاریخ)
